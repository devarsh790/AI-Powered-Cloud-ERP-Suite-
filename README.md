# Amdox ERP: AI-Powered Cloud ERP Suite

![TypeScript](https://img.shields.io/badge/language-TypeScript-3178c6.svg)

A high-fidelity, enterprise-grade ERP platform designed for modern industrial operations. Amdox ERP combines sophisticated engineering aesthetics with integrated AI-driven management.

## 🚀 Key Features

### 🤖 Receptionist Anni (AI Assistant)
- **Personalized Concierge**: A dedicated AI assistant named "Receptionist Anni" to guide users through the workflow.
- **Interactive Assistance**: A fully functional chat interface for contextual insights and rapid operational support.

### 🎨 Premium Interaction Design
- **Tactile UI**: Every interactive element (cards, buttons, tiles) features smooth, hardware-inspired hover animations for a premium feel.
- **Two-Tone Typography**: Standardized enterprise headings with signature Amdox blue highlights for clear visual hierarchy.
- **Industrial Dark Mode**: A deep, glassmorphic UI optimized for industrial tech environments.

### 💼 Integrated Enterprise Modules
The suite provides comprehensive coverage for all business towers:
- 🏦 **Finance**: Accounts Receivable/Payable, General Ledger, and financial reporting.
- 👥 **HR & Payroll**: Automated salary processing, attendance, and employee directories.
- 📦 **Supply Chain**: End-to-end inventory control, purchase orders, and vendor management.
- 🏗 **Projects**: tactical resource allocation and tactical delivery tracking.
- 📊 **Intelligence**: Advanced BI widgets and strategic forecasting.

## 🛠 Tech Stack
- **Frontend**: React 18, Vite, TypeScript.
- **UI/UX**: Custom Vanilla CSS interaction layer, Lucide Icons, Recharts (Data Viz).
- **Backend Architecture**: Node.js, Express, PostgreSQL.

## ⚡ Demo Mode (Quick Start)
The application is pre-configured with a **no-backend demo mode**, allowing you to explore all high-fidelity features immediately without a live database.

1. **Install Dependencies**:
   ```bash
   cd frontend
   npm install
   ```

2. **Run dev server**:
   ```bash
   npm run dev
   ```

3. **Access**:
   Open `http://localhost:5173` and use the default demo credentials.
