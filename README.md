# Amdox ERP: AI-Powered Cloud ERP Suite (Nexus v2)

![React](https://img.shields.io/badge/frontend-React-61dafb.svg)
![TypeScript](https://img.shields.io/badge/language-TypeScript-3178c6.svg)
![Design](https://img.shields.io/badge/design-Nexus%20v2-orange.svg)

A high-fidelity, enterprise-grade ERP platform designed for the futuristic workplace. Amdox ERP (Nexus Edition) combines industrial-tech aesthetics with powerful AI-driven operations.

## 🚀 Key Modernizations

### 🎨 Nexus v2 Design System
- **Premium Aesthetics**: A sophisticated dark-mode UI featuring glassmorphism, depth-based layers, and industrial orange/purple accents.
- **Two-Tone Typography**: Standardized enterprise headings with primary-colored highlights for maximum visual impact.
- **Tactile Interaction Layer**: Every card, tile, and button features smooth, physical-base animations (scaling, elevation, and shadows) on hover for a truly responsive feel.

### 🤖 Receptionist Anni (AI Assistant)
- **Dedicated Concierge**: Rebranded from the "Operations Copilot" to a more personalized AI persona.
- **Interactive Chat**: A fully functional mock-chat interface providing contextual assistance across the platform.
- **Branded Interface**: Custom-logo branding and a clean, high-contrast workspace for AI interactions.

### 📊 Strategic Analytics
- **Enterprise Charts**: Overhauled `Intelligence` module with refined Recharts styling.
- **Operational Clarity**: Removed axis noise, added glassmorphic tooltips, and updated data visualization to match the sleek Nexus palette.
- **Priority Queue**: AI-triaged action items across every module.

## 💼 Comprehensive Modules
The suite covers the full enterprise spectrum:
- 🏦 **Finance**: Invoices, General Ledger (with Indian financial shorthand), Receivables, and Payables.
- 👥 **HR & Payroll**: Employee directory, Attendance, and fully wired Payroll processing.
- 📦 **Supply Chain**: Real-time Inventory tracking, PO management, and Vendor portals.
- 🏗 **Projects**: tactical resource allocation and tactical delivery tracking.

## 🛠 Tech Stack
- **Modern Web**: React 18, Vite, TypeScript.
- **Interactions**: Framer Motion for micro-interactions, custom CSS-based interaction layer.
- **Icons & Data**: Lucide-React, Recharts (Enterprise Grade).
- **Backend Architecture**: Node.js, Express, PostgreSQL (Supporting mock-data fallback for zero-dependency demos).

## ⚡ Getting Started
The project is built to be **demo-ready** immediately without requiring a running database.

1. **Install Dependencies**:
   ```bash
   cd frontend
   npm install
   ```

2. **Run Dev Server**:
   ```bash
   npm run dev
   ```

3. **Access**:
   Open `http://localhost:5173` - Use any demo credentials to access the workspace.
