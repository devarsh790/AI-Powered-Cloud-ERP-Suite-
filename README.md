A high-fidelity, enterprise-grade ERP platform designed for the futuristic workplace. Featuring a custom **Nexus Design System**, glassmorphic UI, and AI-driven analytics.

## ✨ Core Features

- **🚀 Nexus UI Overhaul**: Premium dark-mode aesthetic with electric neon accents and glassmorphism.
- **📊 AI-Powered Analytics**: Real-time insights across HR, Finance, and Supply Chain.
- **🔐 Multi-Tenant Security**: Enterprise-grade authentication and data isolation.
- **💼 Comprehensive Modules**:
  - **HR & Payroll**: Employee management and automated salary processing.
  - **Finance**: General ledger, accounts payable/receivable, and fiscal reporting.
  - **Supply Chain**: Inventory tracking and purchase order management.
  - **Project Management**: Tactical task tracking and resource allocation.
- **🤖 Intelligent Command Palette**: Instant access to all ERP functions via a high-performance search interface.

## 🛠 Tech Stack

- **Frontend**: React 18, TypeScript, Vite, Framer Motion (Micro-interactions), Vanilla CSS (Nexus Design System).
- **Backend**: Node.js, Express, PostgreSQL (with multi-tenant support).
- **DevOps**: Docker, Docker Compose, PowerShell Automation.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- Docker & Docker Compose
- PostgreSQL

### Local Setup
1. **Clone the repository**:
   ```bash
   git clone https://github.com/devarsh790/AI-Powered-Cloud-ERP-Suite-.git
   cd AI-Powered-Cloud-ERP-Suite-
   ```

2. **Run with Docker**:
   ```bash
   docker-compose up -d
   ```

3. **Or run manually using the automation script**:
   ```powershell
   ./start-all.ps1
   ```

## 🎨 Design System
The **Nexus Design System** is built on:
- **Palette**: Deep Navy/Charcoal backgrounds with Electric Cyan (#00F0FF) and Neon Magenta (#FF00EA) accents.
- **Typography**: Outfit for headings, Inter for body text.
- **Aesthetics**: Heavy use of backdrop filters (glassmorphism) and glowing borders.
